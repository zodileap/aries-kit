# 更新记录

本文件根据当前仓库可识别的 Git 发布节点整理。

说明：
- 当前历史里没有独立的 `0.2.2` 发布节点，因此未单独列出。
- `0.2.8` 为当前工作区已完成、准备发布的下一个版本。

## 0.2.8 - 2026-03-15

### 组件与交互

- 为 `Button` 新增 `useColorText` 配置，允许彩色按钮在同色系高对比文字和中性色文字之间切换。
- 调整 `Nav` 容器、logo 与 suffix 的布局约束，保证 `items` 为空或被隐藏时两端内容仍保持贴边。

### 富文本编辑器

- 修复 `RichEditor` 预览容器切换时的引用清理，避免可视模式与预览模式切换后残留旧节点引用。
- 为可视模式补充列表快捷输入能力，输入 `-`、`*`、`+` 或 `1.` 后按空格可自动转换为列表。

### Preview 与文档

- 补充 `Button` 新增属性与 `Nav` 边缘对齐场景的 preview 示例和 API 文档。
- 同步更新 preview 示例源码映射与 props 覆盖审计结果。

## 0.2.7 - 2026-03-11

### 富文本编辑器

- 完善 `RichEditor` 可视编辑能力，修复标题、引用、无序列表、有序列表、任务列表、链接、行内代码、代码块、表格等工具栏行为。
- 补齐图片、视频上传与剪贴板粘贴链路，支持在源码模式和可视模式中插入可保存的媒体标记。
- 增加上传等待能力，保存前可通过实例方法确认并等待待处理媒体上传完成。

### Preview 与组件体验

- 优化 preview 移动端布局，补齐文档页导航、内容区与示例区的小屏适配。
- 修复 preview 根路由未命中问题，并调整 React Router 兼容配置。
- 修复 `DatePicker`、`TimePicker` 禁用态仍显示清除按钮的问题。
- 优化 `Carousel`、`Form` 等组件与示例在移动端下的展示效果。
- 修复 `Code` 组件 Diff 模式整行高亮渲染，并同步更新 preview 文档描述。

### 基础能力

- 为组件库内部补充 i18n 懒初始化，避免宿主应用未显式初始化时富文本等组件报错。
- 移除 `Button` 根节点默认 `margin-bottom`，避免组件在业务页面中自带多余外边距。

## 0.2.6 - 2026-03-10

- 修复 `Nav` 直接点击时的无子菜单交互行为。
- 移除 `List` hover 时的额外位移，统一列表项圆角表现。
